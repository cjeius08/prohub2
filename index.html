Model help

Troubleshooting

Use this when a customer reports no power, clicking, red indicator, filter reset, fan issue, or model-specific questions.

Model steps

Safe troubleshooting flow

  1. 1. Confirm modelAsk for model/SKU if the customer is unsure.
  2. 2. Ask what changedNew filter, moved unit, power outage, unusual sound, light indicator, or no airflow.
  3. 3. Try basic resetPower off, unplug 30–45 seconds, reseat filter/panel, then test again.
  4. 4. Request proof if needed5–8 second video showing the issue and what happens after pressing the button.
  5. 5. Move to warrantyProof of purchase + registration + video if issue remains.
MA-25

Filter reset: press and hold the Dimmer button for 3–5 seconds. Clicking/no lights: run without filter, reseat filter, complete tissue test, and request a 5–8 second video if needed.

MA-35

Fan/no power: power reset, tissue test, run without filter, check for coil whine, and request a 5–8 second video if needed. Auto Mode reads TVOC/PM2.5.

MA-40

No power: check the back panel magnet/latch and complete a 30–45 second power reset. Latest MA-40 reset: hold Replace Filter for 3–5 seconds. MA-40 UV reset: hold Fan + Ionizer for 3–5 seconds.

MA-50 V3

Filter upgrade/fit: insert square, press back upper-left, place unit on side, close until beep, power reset 30–45 seconds, then tissue test. Filter reset: hold Dimmer for 3 seconds.

MA-112

Filter reset: hold Ionizer for 3 seconds. MA-112 Pro has no UV or ionizer. Check model/version before giving UV or ionizer steps.

MA-15 Smart

Red UV/filter indicator: replace filter, then hold Timer + UV for 3–5 seconds. MA-15 Smart has no Dimmer button; UV button doubles as filter indicator.
